# Connection
## GPS - esp32
- VCC -> 3.3V
- GND -> GND
- TX -> RXD
- RX -> TXD

## TFT st7789 -> esp32
- VCC -> 3.3V
- GND -> GND
- TFT_CS  15
- RST -> 4
- DC -> 2
- MOSI -> 23
- MiSO -> 19
- SCK -> 18
- LED -> 3.3V
